Dynamic Path

The Dynamic Path component enhances standard Path functionality and offers greater customizability for your Salesforce processes.

Key Features:

  • 1. Highly Customizable for both Standard and Custom Objects
  • 2. Group multiple picklist values in Path
  • 3. Display Key Fields on record update
  • 4. Remove check mark on completed & Customisable Guidance
